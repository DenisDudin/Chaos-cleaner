# Product Overview

## Vision

Создать простой, интуитивно понятный Mini App для Telegram, который позволяет пользователям быстро получать анализ и сводки из выбранных Telegram‑каналов за любые промежутки времени. Приложение станет персональным "информационным ассистентом", который автоматически собирает данные, обрабатывает их с помощью LLM и предоставляет структурированные результаты в один клик.

## Core Concept

Пользователь выбирает один или несколько каналов, задаёт временной диапазон и формулирует запрос для LLM. Приложение парсит соответствующие посты, передаёт их модели и возвращает удобный, лаконичный вывод — от сводки новостей до тематического анализа.

## Key Features

* **Гибкий сбор данных**: работа с одним каналом или подборками сразу, выбор временных промежутков, быстрый доступ к сохранённым пресетам.
* **Интеграция с LLM**: возможность задавать любой промт для обработки собранного контента — от резюме новостей до аналитики по теме.
* **Сохранённые наборы**: каналы, промты, настройки выборки — всё можно объединять в пресеты и запускать одним действием.
* **Интуитивность**: минимальное количество действий для получения результата, простая логика навигации и взаимодействия.
* **Гибкость**: возможность легко расширять функциональность, добавлять форматы запросов и типы вывода.

## Target Users

* Люди, которые следят за новостями и хотят экономить время.
* Специалисты, которым нужен ежедневный или периодический мониторинг тем (аналитики, редакторы, исследователи).
* Создатели контента, желающие быстро собирать материал из множества источников.

## Main User Flows

* **Быстрый запуск**: выбрать каналы → выбрать период → ввести промт → получить результат.
* **Использование подборок**: выбрать сохранённый пресет → получить обработанный вывод.
* **Создание набора**: добавить каналы → сохранить промт → сохранить как пресет.

## Value Proposition

Приложение устраняет перегрузку информацией, автоматизирует поиск важного и экономит время. Благодаря LLM результаты становятся не просто собранными данными, а готовым к использованию знанием.

## Guiding Principles

* Простота и ясность.
* Минимум действий для пользователя.
* Гибкость без перегруженности.
* Возможность масштабирования и добавления новых сценариев.
